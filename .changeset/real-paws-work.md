first release

Create URL friendly short IDs just like YouTube.

#### Installation

Using npm:

```shell
npm i ytid
```

Using yarn:

```shell
yarn add ytid
```

Using pnpm:

```shell
pnpm i ytid
```

#### Usage

With ES6 (ECMAScript):

```javascript
import { ytid } from "ytid";

console.log(ytid()); // gocwRvLhDf8
```

With CommonJS:

```javascript
const { ytid } = require("ytid");

console.log(ytid()); // dQw4w9WgXcQ
```
